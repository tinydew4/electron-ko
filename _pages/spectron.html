Spectron An Electron Testing Framework

Test your app

Spectron is an open source testing framework for Electron apps. Built on top of ChromeDriver and WebDriverIO.

CI Ready

Tests run on continuous integration services such as Travis and AppVeyor.

Full API

Access the entire Electron API from your Spectron integration tests.

Multi Window

Interact with and verify the behavior of multiple windows from a single test.

Promises

Easily chain together asynchronous operations and assertions.

Extensible

Add additional commands and helpers specific to your app.

Compatible

Can be used with any testing library like Mocha, Jasmine, and AVA.
