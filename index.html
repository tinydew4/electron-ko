Electron

Cross Platform Desktop Application Shell

Formerly known as Atom Shell. Made with by GitHub.

Web Tech

Use HTML, CSS, and JavaScript with Chromium and Node.js to build your app.

Open Source

Electron is open source; mantained by GitHub and an active community.

Cross Platform

Electron apps build and run on Mac, Windows, and Linux.

Built on Electron

With Electron, creating a custom application for your company or idea is easy. Initially developed for GitHub's Atom, Electron has since been used to create applications by companies like Slack, Mapbox and Facebook.

Developers Bring Ideas to Life

Interesting open source tools are built on Electron, too.
  • Hearthdash: Hearthstone tracker
  • Kart: Frontend for RetroArch
  • Monu: Menubar process monitoring app
  • Playback: Experimental video player
  • Screencat: WebRTC screensharing
  • Menubar: Create menubar desktop apps

Get Started with Electron

Getting started with Electron is easy. Download a prebuilt binary or use npm to install from the command line. Full documentaion in the repository.

# Install globally in your $PATH
npm install atom-shell -g

# Install as a development dependency
npm install atom-shell --save-dev
