Built on Electron

With Electron, creating a custom application for your company or idea is easy. Initially developed for GitHub's Atom, Electron has since been used to create applications by companies like Slack, Mapbox and Facebook.

Developers Bring Ideas to Life

Interesting open source tools are built on Electron, too.
  • Hearthdash — Hearthstone tracker
  • Kart — Frontend for RetroArch
  • Monu — Process monitoring app
  • Playback — Experimental video player
  • ScreenCat — WebRTC screensharing
  • Menubar — Create menubar apps

Get Started with Electron

Getting started with Electron is easy. Download a prebuilt binary or use npm to install from the command line. Full documentaion is in the repository.

# Install the `electron` command globally
npm install electron-prebuilt -g

# Install as a development dependency
npm install electron-prebuilt --save-dev
