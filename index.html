Built on Electron

With Electron, creating a desktop application for your company or idea is easy. Initially developed for GitHub's Atom editor, Electron has since been used to create applications by companies like Slack, Mapbox, Docker, and Facebook.

Developers bring ideas to life

Interesting open source tools are built on Electron.
  • Hearthdash — Hearthstone tracker
  • Kart — Frontend for RetroArch
  • Monu — Process monitoring app
  • Playback — Experimental video player
  • ScreenCat — WebRTC screensharing
  • Menubar — Create menubar apps

Get started with Electron

Getting started with Electron is easy. Download a prebuilt binary or use npm to install from the command line. Full documentation is in the repository.

# Install the `electron` command globally
npm install electron-prebuilt -g

# Install as a development dependency
npm install electron-prebuilt --save-dev
